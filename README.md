# 🤖 Reminder Bot

Telegram-бот для напоминаний с поддержкой:
- естественного языка (NLP)
- повторяющихся задач
- часовых поясов
- архитектуры bot + worker + PostgreSQL

---

## 🚀 Возможности

### 🕒 Напоминания
- по дате и времени
- через промежуток времени
- повторяющиеся

### 📌 Примеры
напомни завтра в 9 созвон  
напомни через 30 минут выключить духовку  
напомни каждый день в 10 выпить витамины  
напомни 31.03.2026 18:30 купить молоко  

---

### 🔁 Повторения

Поддерживается:
- каждый день  
- каждую неделю  
- каждый месяц 
- каждый час / каждые X часов
- каждые X минут

---

### 🌍 Часовые пояса
- индивидуально для каждого пользователя  
- формат IANA (Europe/Moscow и т.д.)

---

## 🧠 UX

- /start — кратко + кнопки  
- /help — подробная справка  

Кнопки:
- ➕ Создать напоминание  
- 📋 Мои напоминания  
- 🌍 Часовой пояс  
- ❓ Помощь  

---

## 🏗 Архитектура

Telegram → Bot → PostgreSQL → Worker

---

## ⚙️ Запуск

cp .env.example .env

docker compose build  
docker compose --profile tools run --rm migrate  
docker compose up -d db bot worker  

---

## 📊 Админ

/stats  
/failed  

---

## ⚠️ Важно

- время хранится в UTC  
- пользователю показывается локальное  
- worker обрабатывает задачи
